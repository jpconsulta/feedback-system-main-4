# feedback-system-main-4
This is where feedback is placed to what library services they will avail through a given description and numerical rating. A zip file of feedback-system-main-4 zip is found to download the whole folder compiled with all the files. Furthermore, the goal of this database is to give the users a chance to rate a library system rating from their own experience using the library. They can add descriptions and comments, as well as a rating from 1 to 5. 1 is the lowest which means needs improvement. 5 is the highest, which means excellence.
